감독쌤 Scheduler

브라우저 자동저장 준비 중

입력 및 수정 내용은 자동 저장되나 재부팅할 경우 데이터가 지워질 수 있습니다. 안전한 백업을 위해 큰 수정 전·자동 배정 후·최종 수정 후에는 백업 파일 저장(JSON)을 눌러 파일로 내려받아 보관하세요.
먼저 학교명과 학년별 반 수를 입력합니다.

엑셀에서 표를 복사한 뒤 첫 번째 셀을 클릭하고 붙여넣기 할 수 있습니다. 감독가능은 기본 체크되어 있으며, 체크를 해제하면 모든 감독에서 제외됩니다. 특정 날짜 제외는 날짜 입력란에 2026-04-30, 2026-05-01처럼 쉼표로 구분해 입력합니다. 담당학년은 여러 학년을 체크할 수 있고, 체크된 학년을 기준으로 감독이 편성됩니다.

학년별 감독 수
구분 1학년 정1학년 부1학년 복도 2학년 정2학년 부2학년 복도 3학년 정3학년 부3학년 복도
감독수

화면 상단에서 시험시간·쉬는시간·교시수를 설정합니다. 아래 시트에서 시험날짜만 입력하고 [교시 자동 입력]버튼을 누르면 해당 날짜의 교시와 시작/종료 시간이 자동으로 채워집니다. 교과 칸에서 교과를 선택하면 시험감독이 아닌 수업으로 표시되며, 게시용 감독표에서 교과 정보와 수업 교사를 직접 입력할 수 있습니다. 입력한 교사는 교과 감독 통계에 포함됩니다.

교과로 설정된 경우 담당교사를 수동으로 입력해야 합니다. 게시용 감독표의 시간, 과목과 정/부/복도 감독 이름을 직접 클릭해서 수정할 수 있습니다.

수정 방법: 시간, 과목 또는 교사 이름을 클릭 → 내용 변경 → Enter 또는 다른 곳 클릭. 과목은 같은 날짜·교시·학년 행에 함께 반영되고, 시간은 같은 날짜·교시에 함께 반영됩니다. 감독교사 수정 시 중복 배정 여부를 검사합니다.

목록형 원자료입니다. 이 표에서도 수정할 수 있고, 게시용 감독표의 교사 이름을 직접 클릭해서 수정할 수도 있습니다. 수정 후 통계가 자동 반영됩니다.

머리글의 각 열 이름을 클릭하면 해당 열 기준으로 오름차순/내림차순 정렬할 수 있습니다.

📖 시험감독 편성표 상세 사용 매뉴얼

📂 권장 사용 순서 (전체 워크플로우)

  1. [1. 기본 정보] 탭 → 학교명, 1·2·3학년 반 수 입력
  2. [2. 교사 명단 시트] 탭 → 교사명·교과·담당학년·담임반·제외설정 입력
  3. [3. 시험 시간표 시트] 탭 → 학년도·학기·고사 구분 선택, 날짜/교시/과목 입력
  4. [3. 시험 시간표 시트] → 학년별 정감독·부감독·복도감독 수 설정 후 [랜덤 자동 배정 실행] 클릭
  5. [4. 게시용 감독표] 탭 → 결과 확인, 교사 이름 클릭하여 수동 수정, 🔒 자물쇠로 잠금
  6. [5. 목록형 배정표] 탭 → 원자료 확인·수정, CSV/엑셀 다운로드
  7. [6. 교사별 누적 통계] 탭 → 대시보드 차트로 형평성 확인
  8. 작업 완료 후 [백업 파일 저장(JSON)] 클릭하여 로컬 PC에 저장

🗂️ 탭별 상세 설명

① 기본 정보 탭

  • 학교명: 게시용 감독표 상단에 표시됩니다.
  • 학년별 반 수: 시험실 수 계산의 기준이 됩니다. 예) 1학년 6반 → 1-1 ~ 1-6 시험실 자동 생성.
  • 반 수를 먼저 정확히 입력해야 담임반 선택, 시험실 자동 생성이 올바르게 작동합니다.

② 교사 명단 시트 탭

  • 교사명·교과: 직접 입력하거나 엑셀에서 복사 붙여넣기(Ctrl+V) 가능.
  • 담당학년: 1학년·2학년·3학년·복도 중 복수 체크 가능. 체크 없으면 전 학년 자동 배정.
  • 담임: 본인 담임반을 선택하면 해당 반 고사실 배정에서 자동 배제.
  • 감독가능: 체크 해제 시 모든 배정에서 완전 제외 (예: 병가·출장 중).
  • 제외설정: 요일(월~금)·교시(1~7교시) 체크 및 특정 날짜 텍스트로 감독 제외.
  • CSV 양식 다운로드 → 엑셀 작성 → CSV 업로드로 대량 입력이 편리합니다.

③ 시험 시간표 시트 탭

  • 학년도·학기·고사구분: 상단 드롭다운으로 현재 고사를 선택합니다. 고사별로 시간표가 자동 저장됩니다.
  • 날짜: 2026-04-28 형식으로 입력합니다.
  • 교시: 1교시 ~ 7교시 드롭다운 선택.
  • 1·2·3학년 교과: 각 교시에 실시하는 학년별 과목을 입력합니다. 교과를 입력하면 수업 시간으로 처리되어 통계에서 제외됩니다.
  • 1교시 시작 시각·시험시간·쉬는시간 입력 후 [교시별 시간 자동입력]으로 시작/종료 시간이 자동 계산됩니다.
  • 정감독·부감독·복도감독 수: 학년별로 각 역할에 몇 명을 배정할지 설정합니다.

④ 게시용 감독표 탭

  • 자동 배정 결과를 게시 가능한 표 형태로 보여줍니다.
  • 교사 이름 부분을 직접 클릭하여 다른 교사명으로 수정할 수 있습니다.
  • 이름 옆 🔓/🔒 자물쇠를 클릭하면 잠금 상태로 변경되어, 이후 재배정 시 해당 칸은 변경되지 않습니다.
  • [인쇄/PDF 저장]으로 바로 출력하거나 PDF로 저장할 수 있습니다.
  • 고사 선택 드롭다운으로 이전 고사의 게시용 표도 확인할 수 있습니다.

⑤ 목록형 배정표 탭

  • 날짜·교시·학년·시험실·정감독·부감독·복도감독이 행 단위로 표시됩니다.
  • 이 화면에서도 셀을 직접 클릭하여 수정할 수 있습니다.
  • CSV/엑셀 다운로드로 외부 프로그램에서 추가 편집이 가능합니다.

⑥ 교사별 누적 통계 탭

  • 상단 대시보드에서 최다·최소 감독 교사를 즉시 확인할 수 있습니다.
  • 상위 15명 누적 감독 분포 차트: 정(파란색)·부(보라색)·복도(황토색) 구분 막대그래프.
  • 열 머리글 클릭으로 오름/내림차순 정렬이 가능합니다.
  • 고사가 완료될 때마다 [통계 새로고침]을 눌러 최신 상태로 유지하세요.
  • 연간 통계는 학기별 고사 결과를 자동 누적합니다 (1학기 1차 → 2차 → 2학기 1차 → 2차).

🚫 제외 설정 상세 사용법

요일 체크박스

월·화·수·목·금 중 체크된 요일에는 해당 교사가 자동 배정에서 완전 제외됩니다.
예) 수요일 체크 → 매주 수요일 해당 교사는 배정 불가.

교시 체크박스

1~7교시 중 체크된 교시에는 배정되지 않습니다.
예) 1교시 체크 → 시험 기간 전체의 1교시에 감독 배정 불가.
요일+교시 조합도 가능합니다.

특정 날짜 제외

출장·연가·행사 등 특정 날짜만 배제할 때 사용합니다.
형식: 2026-04-30, 2026-05-01 (쉼표로 구분)
해당 날짜 전체 교시에서 배정 제외됩니다.

🔒 수동 사전 배정(잠금) 기능 사용법

  1. 먼저 자동 배정을 1회 실행하거나, 게시용 감독표에서 교사 이름을 직접 수정합니다.
  2. 고정하려는 교사 이름 옆의 🔓 자물쇠를 클릭하면 🔒 잠금 상태로 변경됩니다.
  3. 이후 [랜덤 자동 배정 실행]을 다시 눌러도 잠긴 칸은 변경되지 않습니다.
  4. 잠긴 교사는 나머지 빈칸 배정 시 당일 누적 횟수에 자동 반영되어 공정한 배분이 이루어집니다.
  5. 다시 🔒를 클릭하면 🔓 잠금 해제 상태로 복구됩니다.

⚙️ 배정 알고리즘 원리

기본 규칙

  • 중복 배정 방지: 같은 날짜·교시에 한 교사가 두 곳 이상 배정되지 않습니다.
  • 담당 학년 우선: 담당학년에 체크된 학년의 고사실에 우선 배치됩니다. 체크 없으면 전 학년 후보입니다.
  • 담임 배제: 본인 담임반 고사실 배정에서 최대한 제외됩니다.
  • 시험 교과 담당 배제: 시험 과목 담당 교사는 해당 고사실 정·부감독에서 제외 후 복도감독으로 우회됩니다.

고도화 규칙

  • 누적 통계 평균화: 이전 고사의 감독 횟수를 기준으로 적게 배정된 교사가 우선됩니다.
  • 연속 감독 방지: 같은 날 동일 교사가 3교시 연속으로 배정되지 않도록 피로도 관리가 작동합니다.
  • 실무사 복도 전용: 교과가 '실무사'인 교사는 복도감독에만 배정됩니다.
  • 복도 전용 교사: 담당학년에 '복도'를 체크한 교사는 복도감독 후보로 우선 분류됩니다.
